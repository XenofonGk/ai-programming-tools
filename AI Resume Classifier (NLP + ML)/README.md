# AI Resume Classifier

A simple NLP-based machine learning project that classifies resume text into job categories using TF-IDF and Logistic Regression.

## Features
- Text preprocessing
- TF-IDF vectorization
- Machine learning classification
- Interactive predictions

## Model Evaluation
The model is evaluated using:
- Accuracy
- Confusion Matrix
- Precision, Recall, and F1-score

This ensures the classifier performance is measurable and interpretable.


## How to Run
```bash
pip install -r requirements.txt
python train.py
python predict.py
